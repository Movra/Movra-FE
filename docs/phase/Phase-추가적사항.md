## 디자인 참고사항

- 기본 폰트는 현재 `BMJUA_ttf.ttf`를 사용한다. 다만 화면 성격이나 가독성에 따라 후속 작업에서 바뀔 수 있다.
- 캐릭터는 흰색에 가까운 둥근 얼굴, 초록 몸체, 머리 위 두 잎, 부드러운 볼터치가 핵심 인상이다.
- 전반적인 톤은 3D 점토/플러시 느낌의 부드러운 질감이며, 날카로운 선이나 강한 대비보다 말랑하고 차분한 입체감을 우선한다.
- 캐릭터 배경은 투명 또는 어두운 배경에서 분리되는 형태로 보이므로, 실제 UI에 배치할 때는 밝은 크림/연녹색 표면 위에 안정적으로 놓는다.
- 로고는 새싹 심볼과 `Movra` 워드마크가 함께 있는 형태를 기준으로 한다.
- 캐릭터는 단순 장식이 아니라 사용자의 현재 행동 상태를 설명하는 시각 신호로 사용한다.

## 화면별 사용 기준

- 기본 캐릭터: 홈, 빈 상태, 일반 안내, 첫 진입 화면.
- MindSweep 중인 캐릭터: 할 일 입력, 계획 정리, 노트/메모 흐름.
- TopPick 고르는 캐릭터: 핵심 행동 선택, 우선순위 결정, 추천 행동 영역.
- 집중하는 캐릭터: Focus Session 시작/진행, 타이머, 집중 상태.
- 작은 성공 캐릭터: 완료, 저장 성공, 작은 성취, 긍정 피드백.
- 다시 일어나는 캐릭터: Recovery Card, 실패 후 복귀, 재시작 안내.
- Movra 로고: 앱 헤더, 로그인 화면, 스플래시/브랜드 노출 영역.

## 이번 작업 기록

- 사용자가 제공한 로고 및 캐릭터 이미지 7종을 디자인 참고 자료로 등록했다.
- 실제 이미지 파일 위치를 확인했다.
- 향후 Phase 작업에서 위 이미지를 UI 참고 기준으로 삼도록 별도 Phase 추가 문서를 생성했다.

## 후속 작업 후보

- 이미지를 프로젝트 내부 정적 자산 경로로 복사할지 결정한다.
- 현재 `SproutMark` SVG와 실제 제공 캐릭터/로고의 역할을 분리할지 결정한다.
- 로그인, 홈 empty state, Focus Session, Recovery Card에 실제 캐릭터 이미지를 적용하는 작업은 별도 Phase 또는 기능 작업에서 진행한다.
